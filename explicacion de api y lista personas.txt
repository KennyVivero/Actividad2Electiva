Esta API, desarrollada con Flask, proporciona un endpoint (`/Personas`) que permite obtener una lista de personas en formato JSON mediante una solicitud `GET`. Cada persona en la lista incluye un nombre y una edad. Para utilizarla, se debe ejecutar el servidor localmente (disponible en `http://127.0.0.1:5000`), y luego realizar una solicitud `GET` a la ruta `/Personas` usando un navegador, herramientas como Postman o comandos como `curl`. La respuesta será un arreglo JSON con los datos de las personas, ideal para integraciones o consultas simples. El servidor está configurado para desarrollo, con recarga automática y mensajes de error detallados.
